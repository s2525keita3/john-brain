# 採用SEO戦略レポート — やるべきこと（2026-06-16）
ぴゅあ訪問看護グループ／専門家6人×3ラウンド議論の統合結論

## エグゼクティブサマリー
- 社長が見ていた「4.6位・37クリック」は**採用LP1枚が拾った"事業検索の残響"**。会社の採用成績ではない。
- **「求人/転職/採用/年収」を含む検索は1クリックも1表示も無い**＝採用LPは順位が低いのではなく**採用市場の土俵に不在**。
- 来ているのは**社名（指名）だけ**。これはほぼ**SNSの遅延した影**で、インスタ95%減により**数か月後に枯れる時限装置**。
- 結論：**①計器を半日で直す ②採用の本丸はGoogleでなく媒体＋スカウト＋TikTok再開 ③Googleは指名防衛としごと検索枠(JobPosting)だけに絞る**。汎用求人ワードのGoogle上位狙いは6人全員「放棄」で合意。

## 2. 掲載順位の信頼性（シークレットウィンドウ問題）
- SCの「平均掲載順位」＝全表示の加重平均＝実在しない合成値。**個別語の今の順位には使えない**。
- シークレットウィンドウ＝自分のIP（地元補正あり）の1サンプル。**両者は次元が違い一致しないのが正常**。
- 罠：①表示10回未満の1位/高CTRは統計的ゴミ→捨てる ②平均順位は前月比トレンドでだけ見る ③地元IPで1位に見えても全国では違う。
- **見るべきは順位でなく imp（高信頼）と CTR（表示10回以上のみ）。第一KPIは「採用クエリのimpが0→1になるか」**。
- 計測スコープ要確認：SCページ別がrecruit.pure-john.comのURLしか返さない→**SC接続先は採用LP単独の公算大**。GASのrowLimitは1000・取得約120行なので「求人クエリ皆無」は打ち切りでなく本物。

## 3. 主要インサイト5点
1. 採用LPは採用市場に不在。直すのは順位でなく「採用語で表示されること」。
2. 指名検索＝SNSの時限装置。SNS回復なくして採用SEOは存続しない。
3. 採用の主戦場はGoogleでない。Googleで効くのは指名防衛とJobPostingの2点のみ。
4. 崩落点は質とクロージング。応募19→入職2。インスタ応募8→入職0、TikTok応募6→入職1。唯一スカウトしたさいわいだけが入職。
5. 桜木町は需要1位だが店舗実体未確認。実店舗なしの桜木町求人ページはJobPosting規約違反。
