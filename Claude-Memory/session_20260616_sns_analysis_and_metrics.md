# 2026-06-16 作業ログ：採用SNS 全量分析と指標整備

「採用データの点検・分析」依頼から、SNS投稿ログ365件の全量分析・専門家パネル提案・指標追加手順書まで実施。詳細な確定事実は [[project-recruit-data-team]] に反映済み。本ログは経緯と成果物の索引。

## 重要な訂正（再発防止）
- 前回の「SNS投稿83日ゼロ🚨」は**誤検知**だった。原因＝`read_file_content`が大きいシート（SNS投稿ログ366行）を3/25で切っていた。
- 正しい読み方：**`download_file_content`(xlsx) → ローカル保存 → openpyxl(data_only=True)で全タブ精読**。シート一覧→実データ行を必ず確認。
- 実際は投稿は6/14まで継続中。リード崩落の起点は**2025年9月**、主因は**インスタのリーチ95%減**（1月4,185→6月204/投稿）。

## SNS分析の核心（投稿365件集計）
- **TikTokは月10万リーチを出すがリンククリック累計=0**（LINE導線が無く認知が換金されていない）。クリックを生む唯一の媒体はインスタ（68件）だがリーチ崩壊。＝2つの蛇口が別々に詰まる。
- **勝ちフォーマット＝「二択×看護師あるある」TikTokリール**（最高52,000「みんなはどっちがいい？」）。スタッフ紹介リール平均10,722。求人臭の「募集・告知」は最低1,985。
- 媒体×タイプ平均リーチ：TikTokリール9,008＞インスタリール2,364＞インスタカルーセル931＞X146＞スレッズ106。**X+スレッズで全投稿の67%(244件)なのにリーチ100前後＝工数と成果が逆相関**。
- カテゴリ：スタッフ紹介2,412が最強。**待遇・条件は保存率最高(4.3)＝転職検討の本命なのに20件しか出していない**。
- インスタはリール(2,364)＞カルーセル(931)なのにカルーセル偏重(28本vs15本)＝リール回帰が回復策。

## 手入力指標の監査（入力率）
- 🔴最重要なのにほぼ空：**平均視聴秒2%・リンククリック数2%**。🟠プロフ閲覧30%・保存17%・フォロワー16%。投稿目的0%。
- 新設すべき列：**フック維持率(最初3秒)・完全視聴率・新規フォロワー獲得数**。

## 指標追加手順書の方針（既存を壊さない）
- SNS投稿ログは現状**A〜T列の20列**。新指標は**U・V・Wに末尾追加のみ**（中間挿入禁止＝ダッシュボード/全体KPI/店舗別の数式参照が壊れるため）。
- 必須2指標(M=リンククリック, S=平均視聴秒)の空欄を条件付き書式で赤表示：`=AND($G2<>"", M2="")`。動画限定なら`=AND($C2="リール", S2="")`。
