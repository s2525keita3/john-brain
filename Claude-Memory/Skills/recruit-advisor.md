## 分析の主軸（SNS採用ファネル）

**採用で本当に効く3層ファネル**

```
第1層：配信量    impressions / reach
  ↓ 視聴率（再生÷表示）が低い → サムネ・冒頭を直す
第2層：視聴維持  views / 完視聴率 / 平均視聴秒
  ↓ 行動率（遷移÷再生）が低い → CTA・締めを直す
第3層：行動転換  profile_tap / followers+ / saves ★最重要
```

**媒体の特性（データで確認済み）**
- Instagram：リーチは小さいが応募転換率が高い。間口広い。
- TikTok：リーチは大きいがプロフィール遷移がほぼ0（計測漏れの可能性あり）。刺さる人には深く刺さる。
- 結論：採用目的ではInstagramが現時点で優位。TikTokは配信層の課題を先に解決する。

**Lステップアンケートの実績（2026年5月時点）**
- 情報収集者323名：Instagram 117名(36%)・TikTok 74名(23%)・未回答87名(27%)
- 転職希望者102名：Instagram・TikTok各21名(同率)・未回答33名
- エントリー58名
- 転職希望者のオンコール：はい57%・相談23%・いいえ3%
- 転職希望者の転職時期：3ヶ月以内32名が最多
- ステーション希望：桜木町1位(26名)・さいわい2位(14名)
